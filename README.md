# パズルでおぼえる「都道府県」

小学校4年生向けに、日本の47都道府県の名称・形・位置をパズル形式で学ぶWebアプリです。

## 起動方法

```bash
npm install
npm run dev
```

ビルド確認:

```bash
npm run build
```

テスト:

```bash
npm test
```

## 実装内容

- 全国モード: 47都道府県ピースを地図へドラッグして配置
- 地方モード: 6地方ごとのスモールステップ学習
- 県庁所在地モード: 県名から県庁所在地を選ぶ4択クイズ
- タイムアタック、カウントダウン、ミス回数、自己ベスト保存
- クリア時の称号、称号コメント、次の称号までの秒数表示
- `100dvh` レイアウト、ページ全体の縦スクロール防止
- SVG `viewBox` による全国表示・地方表示・都道府県フォーカス
- ピース選択時の対象地方への自動フォーカス
- 下部ピーストレイの横スクロール
- localStorage によるモード別・地方別ベストタイム保存

## データ構成

- `src/data/prefectures.ts`: 47都道府県の学習用簡略SVGパス、中心点、県庁所在地
- `src/data/regions.ts`: 指定された6地方区分
- `src/data/capitals.ts`: 県庁所在地クイズ用データ
- `src/data/timeTitles.ts`: タイム称号データ

地図形状は初期MVP向けのオリジナル簡略形状です。既存サイトのコード・画像・地図データは使用していません。

## 地方区分

- 北海道・東北地方: 7
- 関東地方: 7
- 中部地方: 9
- 近畿地方: 7
- 中国・四国地方: 9
- 九州・沖縄地方: 8

北陸地方は単独モードにせず、新潟県・富山県・石川県・福井県は中部地方に含めています。中国地方と四国地方は「中国・四国地方」として統合しています。
